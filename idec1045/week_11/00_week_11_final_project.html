Week 11: Final Project

Introduction

This week has the instructions for the Final project which includes all you have learned this semester. It will become the second part of your professional portfolio. It is a large project and so please start early in the semester to give yourself time to complete it in a timely manner. You will surely be impressed with your end result.

Learning Outcomes

This week you apply all the concepts you have learned over the last several weeks:

  • 1.5 Identify and draw various architectural symbols
  • 2.1 Recognize and apply basic space planning rules
  • 2.2 Identify proper traffic flow
  • 3.3 Draw wall elevations to scale and apply measurements (fold out and separate)
  • 3.4 Develop skills drawing elevation detail, such as fireplaces, doors and windows and furniture
  • 4.1 Apply printing techniques to drawings and legends
  • 4.2 Identify use of pens and drawing tools for professional results.
  • 4.4 Use techniques to transfer drawings to vellum and presentation boards.
  • 4.5 Draw electrical plan and symbols with overlay technique on acetate.
  • 6.2 Interpret required components to create presentation boards
  • 6.4 Develop and apply skills creating presentation boards
  • 6.5 Use various sample techniques for presentation boards
