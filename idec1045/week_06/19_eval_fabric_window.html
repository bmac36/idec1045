Assignment 2: Fabric and Window Treatment

Due: By the last day of the week (11:59 p.m.) of Week 7
Submit to: Fabric and Window Treatment Submission Folder
Value of assignment: 10% of your final grade
Evaluation: Fabric and Window Treatment Rubric

Rationale

  • Outline current fabric trends including pattern, texture and styles as well as uses and benefits of linings to create quality window treatments for various uses
  • Calculate both hard and soft treatments to custom order from a manufacturer or seamstress based on appropriate measurements to complete a window treatment order for a client

Overview

This assignment allows the student to put into practice what they have learned this week. It is a good introduction to what they have to do for the rest of the assignments and final project for the semester.

You will hand in three (3) pages on graph paper, scanned into a word document. Included with the three pages is a working drawing with measurements, written description showing formula to calculate finished length, order yardage, rod length and if necessary pattern repeat, railroading.

Instructions

On a separate page for each of 3 windows:

  • Draw a ½” scale drawing of each window in pencil on graph paper as your working copy
  • On the same page as the drawing (make sure to label each one with the window #):
    • Show the detailed calculations and DESCRIPTION so your instructor can understand what you are calculating of how you came to the fabric total
    • What is the rod width?
    • How much fabric is needed for this treatment (finished length and order/purchase yardage?
    • How much lining is needed for this treatment?
    • Trim is 3” for all assignment (add to the outside of the window measurements)

Window #1:

  1. Ceiling height 10’
  2. Window size 72” W x 76”L (24” from floor to the bottom of the window glass)
  3. Calculate a pair of stationary pinch pleated side panels ( 1 ½ width)
  4. Fabric has 12” pattern repeat
  5. Hung on decorative rod with 2” rings
  6. Panels cover 4” of glass on each side of window
  7. Trouser length on the floor

Window #2:

  1. Ceiling height 8’
  2. Window size 36”W x 66” L (18”from floor to the bottom of the window glass)
  3. Calculate a pair of grommet style operating drapery that cover this window in a plain fabric 2x fullness that covers the window with no stack back.
  4. Calculate how much fabric and lining you would need to order for this treatment
  5. Hang 4” above the window trim on a metal rod using 1 ½” rings
  6. Floor length (1/2” from floor)

Window #3:

  1. Ceiling height 9’
  2. Window size 48” W x 60” L (12” from ceiling to top of window glass)
  3. Calculate a single panel of sheers that do NOT operate but are 1 panel. They will cover the window with no stack back.
  4. Sheers are 3x fullness and determine if the fabric can be railroaded
  5. Hang 6” above the window frame and finish ½“ from the floor
  6. Calculate how much fabric you would need to order for this treatment

Notes

  • Scan each page and add to a single word document which you will submit to the dropbox provided for this assignment.
