Case Study: Operating Treatment

Let's walk through an actual example for operating, or functioning, draperies from start to finish. We'll begin with all the information required to get started, perform all the required calculations and finally order/purchase the fabric and lining for the window treatment.

Part 1: Dimensions

Here are the measurements you need to get started. As we discussed earlier, draw out your window and add your measurements to the drawing before beginning the next step – calculations.

  • Ceiling height: 8'
  • Window size: 5’ (60”) wide x 4’ (48”) high
  • Measurement from window to ceiling: 18”
  • Trim added to outside of window: 6"
  • This is a plain fabric that is 54” wide, with pinch pleats and a standard lining.
  • Rod is a decorative rod with 2” rings.
  • Hang treatment 4” above the trim
  • Draw the window to ½” scale on graph paper for your working copy

Part 2: Calculations

Now that we have all of the measurements, we need to start and have drawn out our sketch of the window and added the measurements to it, it is time find the finished width and length of the fabric, and the length of the rod required.

Width

  • This is one panel on each side of the window so only one piece, or one cut, of fabric is needed
  • 54” fabric minus 6” to finish the salvage edges equals 48” divided by 2.5 fullness for pleated style equals 19.2” taut or 18” relaxed on the rod. This will give 18” width of panel on each side of the window.
  • Rod width: 5’ (60”) for window plus 12” trim (6” on each side of window) plus 8” (4” past the trim on each side) equals 60” plus 12” plus 8” equal 80” finished rod width.
  • Finials will be added to the ends

Length:

  • Measure from floor to top of trim = 78”
  • Add 4” above the trim = 78”+ 4” = 82’= where the rod is hung
  • Rings are 2” diameter so must deduct 2” from the drape length (that hangs down from the rod) = 82-2” = 80”
  • Want ½ “ from the floor for full length = 79 ½” finished length

Part 3: Order Dimensions

The finished lengths of fabric and lining are calculated, but this is not the amount to order. When ordering the fabric, it is necessary to add the fabric needed to make the hem and header. The finished length is 79.5” in length and we need to determine how much fabric is needed for the hem and header.

Here are the things we need to consider:

  • For all pleated drapery always add 16” for hem and header.
  • You need to add 79.5” + 16” = 95.5” minimum length. This is the order length needed to make the panel.
  • The width is already calculated and we know that we need 5 cuts of 54” fabric.
  • 5 cuts X 95.5 length per cut = 477.5”
  • When ordering, you always do so by the yard. There are 36” in a yard so you divide 477.5 by 36 which gives you 13.26 yards.
  • Always round up to allow for errors or flaws in fabric so you would order 13.5 yards minimum of fabric to makes these operating drapes.
  • You will always order by the yard, not inches. In some cases you can also order by the meter which is equal to about 39”.
  • How much to round up? To be safe, always round up to the next quarter yard or more. A bit more fabric will allow for errors or flaws in the fabric and it is always better to have too much fabric than not enough and have to re-order! Remember, you need the same dye lot.
  • Since the fabric has no pattern you would order the same amount of lining.
  • If you had a pattern repeat, you would add this to the length of each cut of fabric to allow matching of pattern.
  • You would adapt the measurements according to the height you want to hang the treatment and how far off the window you want the treatment to be.
